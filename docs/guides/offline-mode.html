Getting Started Command Reference

Offline Mode

Pre-download packages and create portable bundles for air-gapped environments.

Pre-downloading Tools

# Pre-download specific tools $ palm cache fetch aider ollama # Pre-download everything in the registry $ palm cache fetch --all

Downloaded packages are stored in ~/.cache/palm/ and used automatically when installing offline.

Creating Bundles

# Create a portable bundle with all cached packages $ palm cache bundle tools.tar.gz Bundled 5 tools (12.3 MB)

Bundles include the cached packages for all pre-downloaded tools. Transfer the archive to an air-gapped machine and install from it.

Installing Offline

# Install from cache (no network required) $ palm --offline install aider # Install from a bundle $ tar xzf tools.tar.gz -C ~/.cache/palm/ $ palm --offline install aider
Note: Not all install backends support offline mode. pip/uv and docker work well. npm and go may require additional setup.
