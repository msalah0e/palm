Getting Started

Install palm and start managing your AI tools in under a minute.

Installation

Quick Install (recommended)

$ curl -fsSL https://msalah0e.github.io/palm/install.sh | sh

Windows (PowerShell)

PS> irm https://msalah0e.github.io/palm/install.ps1 | iex

With Go

$ go install github.com/msalah0e/palm@latest

From Source

$ git clone https://github.com/msalah0e/palm.git $ cd palm $ make install

Shell Setup

Add completions and vault key exports to your shell:

Zsh (add to ~/.zshrc)

eval "$(palm completion zsh)" eval "$(palm keys env)"

Bash (add to ~/.bashrc)

eval "$(palm completion bash)" eval "$(palm keys env)"

Fish

palm completion fish | source

PowerShell

palm completion powershell | Out-String | Invoke-Expression

This gives you:

  • Tab completion for all commands, tool names, model IDs, and key names
  • Vault key export — all stored API keys are available as environment variables
