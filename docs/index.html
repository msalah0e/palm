What palm Does

Every AI tool has its own installer, config format, and key management. palm unifies all of it into one CLI.

üì¶

Universal Installer

Install any AI tool with one command. Auto-detects the right backend — pip/uv, npm, cargo, go, docker, brew, or script.

palm install claude-code aider ollama
üîê

API Key Vault

Store API keys in macOS Keychain or encrypted file vault. Auto-inject when running tools. No more .env files.

palm keys add ANTHROPIC_API_KEY
üöÄ

Run with Key Injection

Launch any tool with vault keys automatically set as environment variables. Zero config.

palm run aider
üîó

Tool Pipelines

Chain AI tools together. Pipe output from one tool to another with vault keys injected across the entire pipeline.

palm pipe "cat code.py" "|" "aider --fix"
üåê

LLM Proxy

Local API proxy that routes requests to any provider. Auto-injects keys, logs usage, enables budget enforcement.

palm proxy start
üß†

Model Management

Browse 30+ models across 6 providers. Compare pricing, context windows, and capabilities. Pull Ollama models.

palm models list
üí∞

Budget Controls

Set monthly and daily spending limits. Track costs by tool and provider. Never get a surprise bill again.

palm budget set --monthly 50
üìä

Terminal Dashboard

See everything at a glance: installed tools, runtimes, vault keys, LLM providers, budget, sessions, and proxy status.

palm matrix

Benchmark

Run the same prompt through multiple AI tools. Compare response time, quality, and cost side by side.

palm benchmark "explain quicksort"

Workspace Pinning

Pin tools to your project with .palm.toml. Team members run palm workspace install and get the same setup.

palm workspace init

Context Sync

Quick Start

Get up and running in 30 seconds

# Install palm
$ curl -fsSL https://msalah0e.github.io/palm/install.sh | sh
 
# Install your favorite AI tools
$ palm install claude-code aider ollama
 
# Store your API key
$ palm keys add ANTHROPIC_API_KEY
Enter value: sk-ant-api03-...
‚úì Stored
 
# Run a tool with key auto-injected
$ palm run aider

Tool Pipelines

Chain AI tools together

# Pipe code through an AI tool
$ palm pipe "cat main.py" "|" "ollama run llama3.3 'review this'"
 
# Chain multiple tools
$ palm pipe "echo 'write a haiku'" "|" "ollama run llama3.3"
 
# Verbose mode shows each step
$ palm pipe -v "cat README.md" "|" "aider --message 'improve'"
[1/2] cat: 2.1KB
[2/2] aider: processing...
‚úì Pipeline complete 12.3s
