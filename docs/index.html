Why palm

Every AI tool has its own installer, its own config format, its own key management. palm unifies all of it into one CLI.

üì¶

Universal Installer

Install any AI tool with one command. palm detects the right backend — pip/uv, npm, cargo, go, docker, or script.

palm install claude-code
‚ö°

Parallel Install

Install multiple tools concurrently with configurable concurrency. Parallel by default, use --seq to disable.

palm install aider ollama vllm
üîê

API Key Vault

Store API keys securely in macOS Keychain or encrypted file vault. Auto-inject when running tools.

palm keys add ANTHROPIC_API_KEY
üöÄ

Run with Key Injection

Launch any tool with vault keys automatically set as environment variables. No more .env files.

palm run aider
üê≥

Docker Backend

Pull and manage containerized AI tools. Vector databases, inference servers, and more via Docker.

palm install vllm

Offline Mode

All Commands

Everything you need to manage your AI tools.

palm install <tool...>Install AI tool(s) — parallel by default
palm remove <tool>Remove an AI tool
palm update [tool|--all]Update AI tool(s)
palm listList installed AI tools with status
palm search <query>Search the registry by name or tag
palm info <tool>Detailed tool info
palm run <tool> [args...]Run tool with vault keys injected
palm doctorHealth check — tools, keys, runtimes
palm keys [add|rm|list|export]Manage API keys in the vault
palm discoverBrowse curated catalog by category
palm fetch [tool...|--all]Pre-download for offline use
palm bundle <output.tar.gz>Create portable tool bundle
palm self-updateUpdate palm itself
palm statsLocal usage statistics
palm completion <shell>Shell completions (zsh/bash/fish)

Roadmap

From tool manager to AI control plane.

v0.3.0

The AI Tool Manager ‚úÖ

  • Top-level commands (install, remove, update, search, info, doctor)
  • API key vault with macOS Keychain + encrypted file vault
  • palm run with vault key injection

v0.4.0

State & Hooks ‚úÖ

  • State tracking for installed tools
  • External plugin registry (~/.config/palm/plugins/)
  • Lifecycle hooks (pre/post install/run/update)
  • Self-update with GitHub releases version check
  • Multi-tool install, --offline flag, shell installer
  • 102 tools across 13 categories
v0.5.0

Docker + Parallel + Offline ‚úÖ

  • Docker install backend (pull/save/rmi)
  • Parallel multi-tool install with errgroup
  • Offline mode: palm fetch + palm bundle
  • Config hierarchy with project-level .palm.toml
  • Improved pip backend: uv tool > pipx > pip3
v0.6.0

Workspace & Context

  • palm workspace for project-level tool pinning
  • palm context for unified AI tool instructions
  • Shell integration (eval $(palm env))
  • Session history with cost tracking

  • palm budget — spending controls

  • palm models — unified model management
